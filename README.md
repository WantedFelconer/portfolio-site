# WantedFelconer Portfolio

A dark-themed, Material UI-inspired personal portfolio website built with **HTML, CSS, and JavaScript**.  
This site showcases my profile, featured projects, social media presence, and an intro video — all wrapped in a modern, layered design.

---

## ✨ Features

- **Dark Material UI Layout**  
  Sleek, professional design with indigo accents, layered sections, and elevated cards.

- **Avatar + Name Section**  
  Circular avatar displayed beside the name *WantedFelconer*, with designation:  
  *Backend Integration Specialist | Brac University CS Student*.

- **Centered YouTube Intro Video**  
  Embedded video placed perfectly at the center of the page with responsive styling.

- **Featured Projects**  
  Highlighted projects displayed in the center using elevated card components.

- **Animated Social Icons**  
  Footer icons for Facebook, Instagram, LinkedIn, X, YouTube, and GitHub.  
  Each icon animates on hover and opens the respective link in a new tab.

- **Sliding Sidebar Menu**  
  Accessible via a menu icon in the top bar.  
  Sidebar reveals social media links in tile format with hover animations.

## 🚀 Deployment

This portfolio is hosted using **GitHub Pages**.

1. Push the project to your GitHub repository.
2. Go to **Settings → Pages**.
3. Select branch `main` and root `/`.
4. Your site will be live at:
   ```
   https://wantedfelconer.github.io/portfolio/
   ```
